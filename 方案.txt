先直接给结论方向：**我建议用「基于关税‑价格变化的部分均衡 + Armington/Logit 替代模型」来刻画中国在三国大豆之间的进口分配变化，并据此推算美、巴、西三国的大豆出口量和出口值的新分布**；配套的就是一套围绕你给的 USITC 关税数据库 + DataWeb 贸易数据的清洗与匹配流程。下面我分两块说：①建模思路；②具体数据处理方案。

## 一、题意和总体建模思路

题目关键信息：

* 美国、巴西、阿根廷是**全球最大的大豆生产和出口国**，中国是**最大的大豆进口国**。
* 2025 年美国实施“互惠关税”（基线 10% + 对有逆差国家额外 11%–50% 关税），并引发中方反制措施。
* 附件给出了 2020 年以来美国贸易与关税数据（USITC Tariff Database + DataWeb 查询结果），并提供字段说明和代码表。

**问题 1 要求：**

1. 描述中国与美、巴、阿之间的大豆贸易现状；
2. 建立模型评估美国关税调整对三国大豆产业的影响；
3. 估计关税调整后三国大豆出口量与出口值的分布。

核心难点：关税变的是美国对华（以及对其他国家）的关税，但真正直接打击美大豆的是**中国对美报复性关税**，同时美国对其他农产品/投入品加税也会通过成本、汇率等渠道影响世界价格。本题在比赛框架下没法做全局一般均衡，只能做**以中国市场为中心的部分均衡 + 结构化贸易（替代）模型**。

所以我建议整体结构：

1. **现状分析模块**：用描述性指标刻画中国自三国的大豆进口结构。
2. **关税–价格传导模块**：把美国“互惠关税”和中方反制映射成各国对华大豆出口的到岸价变化。
3. **进口来源替代模型（Armington/Logit）**：在给定中国大豆总需求的前提下，按价格变化重新分配从美国、巴西、阿根廷的进口份额。
4. **产业影响模块**：用出口量和出口值变化衡量对三国大豆产业的冲击（可再加一个简单供给弹性来估计产量变化）。

### 2. 大豆商品编码识别（HTS/HS 对应）

处理建议：

1. 在关税数据库 `trade_tariff_database_YYYYMM.txt` 中，用 `brief_description` 搜索含 “SOYBEAN” / “SOYA-BEAN” 等关键词的行，列出对应的 `hts8` 码。

### 3. DataWeb 贸易 Excel 的整理方法

两份 excel 的结构类似（上面我已经用 Python 看过）：

#### 3.1 FAS Value（出口）

`DataWeb-Query-Export.xlsx` – sheet: **FAS Value**

前几行格式大致是：

| 行 | Total Exports|Annual Data | Unnamed:1  | Unnamed:2    | Unnamed:3   | Unnamed:4…    |
| - | ------------------------- | ---------- | ------------ | ----------- | ------------- |
| 0 | Data Row Count            | 17773      |              |             |               |
| 1 | Data Type                 | HTS Number | Description  | Country     | 2020 2021 ... |
| 2 | FAS Value                 | 01         | LIVE ANIMALS | Afghanistan | 4483 ...      |

清洗步骤（可以在 Excel/或 Python 中实现）：

1. **删除第 0 行**（Data Row Count）；

2. 将第 1 行作为列名，重命名为更友好的字段，例如：

   * `Data Type` → `data_type`
   * `HTS Number` → `hts`
   * `Description` → `description`
   * `Country` → `country`
   * 年份列 `2020`–`2025` 保持即可

3. 过滤：`data_type == 'FAS Value'`。

4. 如果只用 2 位 HTS：筛选 `hts == '12'` 代表油籽章节；若你改做更细查询，就筛 `hts` 以 `1201` 开头。

5. 筛选国家：`country ∈ {China, Brazil, Argentina, World}`（世界总量可用来算占比）。

6. **拉长为长表（long format）**：把 2020–2025 列用“年–值”两列表示，得到结构例如：

   | hts | description | country | year | export_fas_value |
   | --- | ----------- | ------- | ---- | ---------------- |

7. 单位：FAS Value 是美元金额（出口货值），默认不用再换单位；为了方便计算，可以统一换成亿/十亿美元。

这部分给你：

* 美国对中国、巴西、阿根廷的出口结构（虽然问题 1 更关心对中国出口）；
* 也能看美国大豆对世界出口的比重和趋势，为后面“产业影响”做背景。

#### 3.2 General Import Charges（进口关税收入）

`DataWeb-Query-Import.xlsx` – sheet: **General Import Charges**

结构类似，清洗完全同 3.1：

1. 删除第 0 行，重命名第 1 行为列名。
2. 保留 `data_type == 'General Import Charges'` 的记录。
3. 同样根据 `hts` 和 `country` 筛大豆相关章节 / 国家。
4. 拉成长表，得到：

   | hts | description | country | year | general_import_charges |

> 这一列是**关税收入**（不是税率），用来近似推算美国对某章货物的平均有效税率：
>
> [
> \text{avg_tariff}*{US, hts, year}
> \approx
> \frac{\text{General Import Charges}*{hts,year}}{\text{Import Value}_{hts,year}}
> ]
>
> 其中分母的 Import Value 需要再做一份 DataWeb 查询（General Imports 的 customs value），题目允许你自行补充数据来源。

这部分可在问题一中用于说明：**美国对农产品/油籽原本整体关税水平很低**，但“互惠关税”会把它整体抬高，对世界价格形成压力。

### 4. USITC Tariff Database 的处理

`tariff_data_YYYY.zip` 中的 `trade_tariff_database_YYYYMM.txt` 是列式 csv，大约 120 列，字段含义详见 `td-fields.pdf`。

#### 4.1 过滤出大豆相关税目

1. 读取每一年（或关键年份，如 2020、2024）的 txt：

   * 关心字段：`hts8`, `brief_description`, `mfn_rate_type_code`,
     `mfn_ad_val_rate`, `mfn_specific_rate`, `mfn_other_rate`,
     `col1_special_text`, 各类 FTA 指示（如 `USMCA_INDICATOR` 等）。
2. 用 `hts8` 前四位属于 1201/1208/2304/1507 的行构造“大豆组”；
3. 若你只想聚焦原粮大豆，可只保留 1201.*。

#### 4.2 计算基准 MFN 税率

利用 `mfn_rate_type_code` 和 `Code Key` 来解读关税类型：

* 常见情况：

  * `mfn_rate_type_code == 0` → MFN “Free”，税率为 0；
  * `mfn_rate_type_code == 7` → 纯从价税，税率在 `mfn_ad_val_rate` 中（如 0.05=5%）；
  * `1,2,3,4,5,6` 涉及从量税/复合税，需要结合 `mfn_specific_rate`、`quantity_1_code` 等，但大豆通常是 Free 或低从价税，你可以在数据中确认。

对每个大豆税目构造：

[
t^{MFN}_{US, h} =
\begin{cases}
mfn_ad_val_rate, & \text{若 } mfn_rate_type_code = 7\
0, & \text{若 MFN_TEXT_RATE = Free 或 code=0}\
\text{按 Code Key 换算}, & \text{其他情况}
\end{cases}
]

其中 h 表示具体的 8 位税目。

> 这一步主要是为了在模型里设定**基准关税水平**，然后叠加“互惠关税”情景。

## 三、数学建模方案

### 1. 现状分析模块

**目标**：定量刻画目前中国自美、巴、阿的大豆进口格局，以及三国在全球大豆贸易中的位置。

#### 1.1 主要指标

1. **中国对各国大豆进口额与数量**

   [
   X^{0}*{i\to CN}, \quad Q^{0}*{i\to CN}
   ]

2. **中国自各国大豆进口占比**（按数量或金额）

   [
   s^{0}*{i,CN} = \frac{X^{0}*{i\to CN}}{M^{0}_{CN}}
   ]

3. **三国对华出口在本国大豆总出口中的比重**

   [
   \theta^{0}*{i, CN}
   = \frac{X^{0}*{i\to CN}}{X^{0}_{i\to World}}
   ]

   其中 (X^{0}_{i\to World}) 可由各国出口统计或 DataWeb（对美国）得到。

4. **市场集中度指标**（如 Herfindahl–Hirschman 指数）

   [
   HHI_{CN} = \sum_i (s^{0}_{i,CN})^2
   ]

用这些指标画时间序列图（2020–2024），就能完成“当前贸易现状分析”部分。

---

### 2. 关税–价格传导模块

这一模块把美国“互惠关税”以及中方反制措施转化为**中国进口各来源大豆的到岸价变化**。

#### 2.1 中国进口端价格

设：

* (i \in {US,BR,AR})；
* (p_i^w)：i 国大豆世界市场 FOB 单价（USD/吨）；
* (c_{i\to CN})：运费等基准贸易成本；
* (\tau^{CN}_i)：中国对来自 i 的大豆进口 ad valorem 税率（基准）；
* (\Delta\tau^{CN}_i)：由于报复性关税引起的增量（对美 > 0，对巴/阿≈0）。

则中国进口 i 国大豆的含税到岸价：

* **关税调整前**：

  [
  P^{0}*{i\to CN} = (p_i^w + c*{i\to CN}) \cdot (1 + \tau^{CN}_i)
  ]

* **关税调整后**：

  [
  P^{1}*{i\to CN} = (p_i^w + c*{i\to CN}) \cdot (1 + \tau^{CN}_i + \Delta\tau^{CN}_i)
  ]

#### 2.2 美国“互惠关税”对世界价格的影响（简化处理）

严格来说，美国抬高对其他国家农产品和工业品的关税，会通过以下渠道改变世界大豆价格：

* 减少美国对巴/阿等国农产品进口 → 这些国家更多出口面向中国；
* 中国对美反制关税 → 减少中国自美进口，压低美国农户的大豆价格；
* 这些效果综合起来，会导致 (p_i^w) 的变化。

由于题目数据有限、模型复杂，可以采用**情景设定**简化：

* 情景 1：世界大豆基准价格不变（短期供给刚性），即 (p_i^w) 不变，只通过关税改变中国到岸价；
* 情景 2：给定一个价格调整幅度，例如：美国农户因被中国限制进口，国内大豆价格下跌 5%–10%；巴/阿因向中国出口增加，价格小幅上升 2%–3%，即：

  [
  p^{w,1}*{US} = p^{w,0} (1 - \delta*{US}),\quad
  p^{w,1}*{BR} = p^{w,0} (1 + \delta*{BR}),\quad
  p^{w,1}*{AR} = p^{w,0} (1 + \delta*{AR})
  ]

这样你就可以叠加关税 + 世界价格两种效应。

---

### 3. 中国进口来源替代模型（Armington/Logit）

**核心思想**：中国把来自不同国家的大豆视作“同类但可替代”的差异化商品，其从每个来源国的进口量取决于：

* 总体大豆需求规模；
* 各来源国到岸价（含关税、运费）；
* 来源国的“偏好/品质”参数。

这就是经典的 **Armington 偏好假设**。

#### 3.1 变量与参数

* (M_{CN})：中国大豆进口总量；
* (X_{i\to CN})：中国自 i 的大豆进口量（模型求解的核心）；
* (s_{i,CN} = X_{i\to CN} / M_{CN})：中国自 i 的进口份额；
* (P_{i\to CN})：前面算的到岸价（0/1 时段）；
* (\sigma)：来源国替代弹性（大于 0，通常 3–7 之间）；
* (\alpha_i)：来源国 i 的偏好参数（由基期数据校准）。

#### 3.2 Armington 份额公式

假设中国的**大豆进口需求函数**为 CES 形式，导出来源份额（不展开推导）：

[
s_{i,CN} =
\frac{\alpha_i , P_{i\to CN}^{-\sigma}}
{\sum_{k} \alpha_k , P_{k\to CN}^{-\sigma}}
]

在**基准期（关税调整前）**，我们有观测值：

* (s^{0}_{i,CN})（基于真实进口数据求得）；
* (P^{0}_{i\to CN})（按上节计算）。

给定替代弹性 (\sigma)，可用基期份额来反推出 (\alpha_i) 的相对值，例如设：

[
\alpha_i = s^{0}*{i,CN} (P^{0}*{i\to CN})^{\sigma}
]

再统一除以某个常数即可（因为只看相对比重）。

#### 3.3 关税调整后的份额和进口量

1. **更新价格**：用新的关税和世界价格得到 (P^{1}_{i\to CN})。
2. **计算关税后份额**：

[
s^{1}*{i,CN} =
\frac{\alpha_i , (P^{1}*{i\to CN})^{-\sigma}}
{\sum_{k} \alpha_k , (P^{1}_{k\to CN})^{-\sigma}}
]

3. **中国大豆总进口量的调整**（可选）：

可以假设中国大豆总需求对大豆综合价格 ( \bar{P}_{CN}) 具有价格弹性 (\eta<0)，例如：

* **综合价格**：

  [
  \bar{P}^0_{CN} = \sum_i s^{0}*{i,CN} P^{0}*{i\to CN},\quad
  \bar{P}^1_{CN} = \sum_i s^{1}*{i,CN} P^{1}*{i\to CN}
  ]

* **总需求变化**：

  [
  M^{1}*{CN} = M^{0}*{CN}
  \left(\frac{\bar{P}^1_{CN}}{\bar{P}^0_{CN}}\right)^{\eta}
  ]

  其中 (\eta) 可设为 -0.3 左右（短期粮食需求较刚性），并在敏感性分析中取多个值。

4. **得到各国新进口量**：

[
X^{1}*{i\to CN} = s^{1}*{i,CN} \cdot M^{1}_{CN}
]

5. **出口值**（以 FOB 价或 CIF 价计）：

* 以出口方视角（FOB）：

  [
  V^{1}*{i\to CN} = X^{1}*{i\to CN} \cdot p^{w,1}_i
  ]

* 以中国视角（到岸价）：

  [
  V^{1, CIF}*{i\to CN} = X^{1}*{i\to CN} \cdot P^{1}_{i\to CN}
  ]

问题 1 要求的是“出口量和出口值的分布”，你可以主要给出 (X^{1}*{i\to CN}) 和 (V^{1}*{i\to CN})（以美元计），同时给出份额 (s^{1}_{i,CN}) 比较关税调整前后的变化。

---

### 4. 出口国大豆产业影响评估

虽然问题 1 只要求三国出口分布，但从“产业影响”角度，至少可以做三层分析：

#### 4.1 出口量与出口值变化

对每个国家 i：

* 对华出口量变化：

  [
  \Delta X_{i\to CN} = X^{1}*{i\to CN} - X^{0}*{i\to CN}
  ]

* 对华出口值变化：

  [
  \Delta V_{i\to CN} = V^{1}*{i\to CN} - V^{0}*{i\to CN}
  ]

* 若能估算 i 国对世界总出口对中国的依赖度 (\theta_{i,CN})，可近似总出口变化：

  [
  \Delta X_{i, World}
  \approx \frac{\Delta X_{i\to CN}}{\theta^{0}_{i,CN}}
  ]

  （假设对中国的出口变化按比例反映在总出口中）

#### 4.2 农户收益（价格 × 数量）的近似

如果有国内农场价格 (p^{farm}_i)，或者假设出口价格变化可以近似代表农户收入变化，可以估计：

[
\Delta R_i \approx \Delta X_{i, World} \cdot p^{farm}_i
]

或用相对变化：

[
\frac{\Delta R_i}{R_i^0}
\approx
\frac{\Delta X_{i, World}}{X_{i, World}^0}
+
\frac{\Delta p^{farm}_i}{p^{farm}_i}
]

其中 (\Delta p^{farm}_i) 可以在前面的情景假设中给出（例如美国因失去中国市场而价格下跌 5%）。

#### 4.3 简单供给弹性模型（可选加分）

若你希望把出口变化延伸到“产量变化”，可以假设各国大豆供给函数为：

[
Q^S_i = A_i , (p^{farm}_i)^{\varepsilon_i}
]

其中 (\varepsilon_i) 为供给弹性（>0），A_i 为常数。

在关税冲击前后，给定价格变化 (p^{farm,0}_i \to p^{farm,1}_i)，得到产量变化：

[
\frac{Q^{S,1}_i}{Q^{S,0}_i}
===========================

\left(\frac{p^{farm,1}_i}{p^{farm,0}_i}\right)^{\varepsilon_i}
]

这样就能把贸易冲击和产业层面的产量/种植面积联系起来（可在论文中做定性讨论或简单量化）。

## 四、模型求解流程小结

你可以在论文中按“算法流程图”的方式呈现：

1. **数据准备**

   1. 从中国海关/UN Comtrade 获取基期下中国自美、巴、阿大豆进口的数量与金额。
   2. 从 DataWeb（或各国统计）获得三国对世界大豆出口数据。
   3. 用 USITC Tariff Database 提取大豆相关 HTS 项下的 MFN 和优惠税率，构造基准税率；结合题意设定“互惠关税”及中国报复关税情景。
   4. 计算基期各来源国对华大豆到岸价 (P^{0}*{i\to CN}) 和份额 (s^{0}*{i,CN})，以及综合价格 (\bar{P}^0_{CN})。

2. **参数校准**

   1. 给定来源国替代弹性 (\sigma) 和需求弹性 (\eta) 的初值（可参考文献或情景设定）。
   2. 用基期 (s^{0}*{i,CN})、(P^{0}*{i\to CN}) 计算 Armington 偏好参数 (\alpha_i)。

3. **政策情景设定**

   1. 根据题意设定美国“互惠关税”与中方反制关税（对美大豆加征 X%，对巴/阿关税不变或小幅调整）。
   2. 设定世界大豆价格的变化情景（不变/美国跌价/巴西阿根廷升价）。

4. **关税冲击模拟**

   1. 计算新到岸价 (P^{1}_{i\to CN})。
   2. 用 Armington 模型算出新份额 (s^{1}_{i,CN})。
   3. 用价格弹性 (\eta) 更新中国大豆总进口量 (M^{1}_{CN})。
   4. 得到新的进口量和出口值 (X^{1}*{i\to CN}, V^{1}*{i\to CN})。

5. **结果分析**

   1. 比较关税前后各国对华出口量/值的绝对变化和相对份额变化；
   2. 推算对各国大豆产业的收入和产量影响；
   3. 做敏感性分析（改变 (\sigma)、(\eta)、关税幅度等），评价结果的稳健性。
