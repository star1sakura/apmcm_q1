## 1. 直接给出“供给弹性 / 需求弹性”区间的文献

在《国际大豆市场对中国大豆市场尾部风险溢出研究》（章少宾，博士论文）里，作者在讨论**国际价格通过贸易传导到中国市场**时，明确写到：

> 贸易传导强度受进口依存度、**贸易弹性系数（供给弹性 0.7–1.2，需求弹性 0.3–0.8）**及全球市场份额等参数综合影响（Zhang et al., 2024）。

这段基本就是你模型里 η 的典型取值范围：

* 供给价格弹性：**0.7–1.2**
* 需求价格弹性（绝对值）：**0.3–0.8**

如果你模型里的 η 是“价格变动 1% 引起数量变动的百分比”（需求或供给价格弹性），可以这样用：

* **基准情景**：

  * 需求弹性取中位值：η_d ≈ 0.5
  * 供给弹性取中位值：η_s ≈ 1.0
* **敏感性分析**：

  * 需求：0.3 / 0.8 两端各跑一次
  * 供给：0.7 / 1.2 两端各跑一次

在论文里写的时候可以直接说明：
“参考章少宾（2024）对粮食贸易弹性的综述，假定大豆需求价格弹性绝对值在 0.3–0.8 之间、供给价格弹性在 0.7–1.2 之间，本文基准情景取 η_d=0.5，η_s=1.0，并进行区间敏感性分析。”——这样引用是完全说得过去的。

同一篇论文里还给了一个**极端刚性需求**的数值：

> 饲料加工企业的需求刚性（短期需求弹性仅 0.15）。

你可以把 **0.15** 当成“短期极端刚性”的下界情景，用来说明：即便需求几乎不动，结论是不是还稳得住。

> 💡 建议：
>
> * 如果你模型里 η 是“需求价格弹性绝对值”，可以设置三种情景：0.15（极刚性）、0.5（基准）、0.8（相对富有弹性），对比关税冲击下出口量 / 进口量的变化。

## 2. 用引力模型回归系数当“贸易弹性”

《基于贸易引力模型的中国与大豆主产国贸易影响因素分析》（芦霁嘉，硕士）虽然没有直接写“弹性”，但因为是**log-log 的引力模型**，很多系数本身就是“弹性”。

例如文中提到：

> 中国大豆生产者价格增加 1%，双边贸易额随之增加 1.098%，并在 5% 水平上显著。

这里这个 **1.098** 就是：

> “中国—某主产国大豆双边贸易额 对 中国大豆生产者价格 的弹性”。

如果你问题一的模型里设的是**“双边贸易量对价格/成本（含关税）变化的弹性 η”**，完全可以把这类系数理解为：

> η_trade ≈ 1 左右

所以：

* 若 η 表示“价格（含关税）↑1%，出口量 / 贸易量变化 η%”，可以用：

  * 基准：η_trade ≈ 1.0
  * 范围：0.7–1.2（和上面供给弹性接近，也有文献支持）

## 3. 替代弹性 / 其他弹性

几篇文献会提到“替代弹性高 / 供给弹性不足”等，但多数是定性，没有具体数值，比如：

* 指出豆粕替代品“供给弹性不足”“替代空间有限”；
* 指出植物油之间“具有显著的需求替代弹性”，但没给具体替代弹性系数。

所以：

* **价格弹性（供给/需求）**：可以直接用上面给出的区间；
* **跨品种替代弹性（比如大豆油 vs 棕榈油）**：这些文献主要是定性描述，没给明确数值，如果模型不强依赖这个参数，可以简化为“有一定替代性 / 完全刚性”两种情景即可。

## 4. 怎么在你的 Q1 模型里“落地”这些 η

结合你现在做的问题一，可以这么操作：

1. **先明确 η 的含义**

   * 如果你模型是：
     ( Q_d = Q_{d0} (P/P_0)^{-\eta_d} )，那 η_d 就是**需求价格弹性绝对值**。
   * 如果你写的是：
     ( \ln X_{ij} = \dots - \eta \ln(1+\tau_{ij}) )，那 η 就是**贸易流量对关税（价格）变化的弹性**。

2. **选取带文献支持的基准值**

   * 需求价格弹性（进口国）：η_d ≈ 0.5（基准），区间 [0.3, 0.8]；极端刚性可用 0.15。
   * 供给价格弹性 / 贸易弹性：η_s 或 η_trade ≈ 1.0，区间 [0.7, 1.2]。

3. **写在论文里的时候记得注明**：

   * “参数取值参考某博士论文及引力模型实证研究，并通过区间敏感性分析检验结论稳健性”，这样评委会看得出你不是拍脑袋，而是有文献依据 + 做了稳健性检验。
